Shift Puzzle

Description

As the top diagram illustrate's this is a fun little circuit, using a shift register, some detection logic of the correct sequence, then a latched output to display the binary equvilent of any 3digit decimal code ( For the padlock ), and yes it is a very slow clock (0.5-1.5Hz) no worries with any race condition or frustrated players ^^.
